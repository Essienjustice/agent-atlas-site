# Agent Atlas — Marketing Site

Landing page for Agent Atlas, the on-chain reputation protocol for autonomous AI agents built on Mantle.

**Live:** https://agent-atlas-site.vercel.app

## About

This is the marketing and documentation site for Agent Atlas. It features:
- Live protocol metrics fetched from the real backend API
- How It Works walkthrough
- Contract addresses and explorer links
- Full documentation

## dApp

The actual protocol dApp is at: https://agent-atlas-tau.vercel.app

## Monorepo

The main protocol monorepo (contracts, backend, indexer, frontend) is at:
https://github.com/Essienjustice/agent-atlas

## Tech Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Running Locally

```bash
git clone https://github.com/Essienjustice/agent-atlas-site
cd agent-atlas-site
npm install
npm run dev
```
